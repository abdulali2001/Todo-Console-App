# Todo App

A minimal, clean CLI todo manager using Python 3.13+.

## Features

- Add new todo items with title and optional description
- View all todo items with status indicators
- Mark todo items as complete/incomplete
- Update existing todo items
- Delete specific todo items
- Clear all todo items with confirmation
- In-memory storage (no persistence)

## Usage

Run the application with:

```bash
python -m src.todo.main
```

## Requirements

- Python 3.13+
- No external dependencies
